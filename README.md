Данный проект был создан в процессе выполнения лабораторной работы по информационной безопасности (2курс) по теме кибермошенничество. 
Целью проекта было создать фишинговую рассылку, которая бы собирала данные пользователей (пароль, логин), маскируясь под почтовое сообщение от поликлиники.
